API calls /getProjectDetails, /getClientName, /getClientEmail and /getProjectURL must be suffixed with ?project_name=PROJECT_NAME where PROJECT_NAME is the exact name of an existing project. Spaces in PROJECT_NAME must be replaced with the + symbol. All existing projects are returned and can therefore be viewed by the /getAllProjects API call.
